# Socket Game

# Requirements to Run
1. Python3

# Packages to Run
No additional packages are required

# Test Cases
No test cases are present as of now

# Instructions to Run
1. Export the port number to run the server using:
    `export SOCKET_GAME_PORT=8000`
2. Run the server using:
    `python3 server.py`
3. Run the client using:
    `python3 client.py`
